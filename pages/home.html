Topic 05 : Referrer-based access control

This demo web application was created for educational purposes to simulate a security vulnerability in access control based on the HTTP Referer header.

The objective of this assignment is to demonstrate how relying on the Referer header to protect sensitive functionalities, such as admin access, can be bypassed by an attacker to escalate privileges and gain unauthorized access.
